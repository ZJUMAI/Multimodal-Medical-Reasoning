# Images for A Survey of Medical Reasoning Foundation Models

This directory contains figures and visualizations for the survey repository.

## Required Images

Please add the following images to complete the repository:

1. **overview.png** - Survey overview diagram showing the hierarchical organization
   - Suggested size: 800px width
   - Should illustrate the main components of the survey

2. **roadmap.png** - Detailed roadmap of medical reasoning approaches
   - Suggested size: 900px width
   - Should show the progression from LLMs to multimodal reasoning

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: High resolution for clarity (at least 1200px width)
- **Style**: Professional, academic presentation style
- **File naming**: Use lowercase with hyphens (e.g., `overview-diagram.png`)
